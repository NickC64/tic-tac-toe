# Example Theme Files

This directory contains example theme files that you can import to test the theme system. Each theme has a unique color scheme and font combination.

## Available Example Themes

### 1. Purple Dream Theme (`example-theme-purple.json`)
- **Colors**: Vibrant purple color scheme
- **Fonts**: Poppins (body), Merriweather (headings)
- **Style**: Modern and elegant

### 2. Ocean Breeze Theme (`example-theme-ocean.json`)
- **Colors**: Calming blue-green ocean-inspired colors
- **Fonts**: Nunito (body), Lora (headings)
- **Style**: Fresh and calming

### 3. Sunset Warmth Theme (`example-theme-sunset.json`)
- **Colors**: Warm orange-red sunset colors
- **Fonts**: Montserrat (body), Crimson Text (headings)
- **Style**: Cozy and warm

## How to Use

1. Go to the **Settings** page in the application
2. In the **Theme** section, click **Import Theme File**
3. Select one of the example theme files from the `public` folder
4. The theme will be imported and automatically applied
5. Use the theme toggle button to switch between light and dark variants

## Testing Tips

- Try importing different themes to see the color and font changes
- Toggle between light and dark modes to see both variants
- Check the **Font Information** section in Settings to see which fonts are loaded
- Export a theme after importing to see the full theme file structure

## Creating Your Own Theme

You can use these examples as a starting point for creating your own themes. Simply:
1. Copy one of the example files
2. Modify the colors, fonts, and metadata
3. Import your custom theme file

For more information, see `THEME_FILE_FORMAT.md` and `FONT_SYSTEM.md`.
